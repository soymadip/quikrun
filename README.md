  <h1>Quikrun</h1>
  <h4>Run Your code without hassle</h4>
  <p>Quikrun is a CLI tool to run code files during learning/teaching without the hassle of manually typing the entire command in terminal</p>
</div align="center">

## Installation

### Distro Packages

#### Arch Linux (AUR)

```bash
paru -S quikrun
```

### From PyPI

#### Using UV

```bash
uv tool install quikrun
```

#### Using Pip/Pipx

```bash
pip install quikrun

# Using pipx (recommended instead of pip)
pipx install quikrun
```

### Using Git

```bash
git clone https://github.com/soymadip/quikrun
cd quikrun

./bin/quikrun -h
```
